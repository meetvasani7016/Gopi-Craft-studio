# Gopi Craft-Studio: Where Tradition Meets Elegance

Gopi Craft-Studio is a premium, production-grade e-commerce platform specializing in handcrafted traditional Indian decor, pooja accessories, and heritage artifacts. 

The application is built on **Next.js** (App Router), styled using **Vanilla CSS**, and backed by **Supabase** for database, file storage, and authentication.

## 🛠️ Main Features

- **Setup Wizard**: Easy, secure registration of the first Super Admin account.
- **Inventory & Variant Engine**: Auto-deductions during checkout, variant SKU/price overrides, and color-coded stock warnings (Green/Orange/Red).
- **Dynamic Shipping**: Rules engine calculating shipping costs by Indian state codes (with free shipping thresholds).
- **Administrative Logs**: Audit log tracking administrative edits, logins, and configurations updates.
- **Media Alt Tags**: Alt text mapping for images, renaming, and image replacements directly in the media panel.
- **CSV Catalog Tools**: Bulk import and export for products and categories.
